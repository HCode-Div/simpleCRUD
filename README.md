# simpleCRUD
SimpleCRUD is a simple web application built with Angular that provides a clean and organized interface to manage data easily. It allows viewing data quickly and efficiently with an interactive and smooth user experience, without direct add, edit, or delete functionalities.

SimpleCRUD is a web application built with Angular featuring a login system that accepts valid email addresses (e.g., example@mail.com
) and passwords of at least 3 characters. Once logged in, users can access a dashboard to add new items, view them, and delete them. Editing and updating items will be available in an upcoming update. The app provides a clean and interactive interface for managing data efficiently.
